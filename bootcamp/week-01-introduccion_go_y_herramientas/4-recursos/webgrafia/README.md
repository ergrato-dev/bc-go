# Recursos — Webgrafía

Documentación y artículos online recomendados para la Semana 01.

---

## 🌐 Documentación oficial

| Recurso | URL | Para qué |
|---------|-----|----------|
| Go Documentation Hub | [go.dev/doc](https://go.dev/doc/) | Punto de entrada a toda la documentación oficial |
| A Tour of Go | [go.dev/tour](https://go.dev/tour/) | Tutorial interactivo en el navegador — sin instalar nada |
| Effective Go | [go.dev/doc/effective_go](https://go.dev/doc/effective_go) | Guía de estilo idiomático escrita por el equipo de Go |
| Go Specification | [go.dev/ref/spec](https://go.dev/ref/spec) | Especificación formal del lenguaje (referencia, no tutorial) |
| Go Modules Reference | [go.dev/ref/mod](https://go.dev/ref/mod) | Todo sobre go.mod, go.sum y el sistema de módulos |

## 🌐 Herramientas online

| Recurso | URL | Para qué |
|---------|-----|----------|
| Go Playground | [go.dev/play](https://go.dev/play/) | Ejecutar Go en el navegador, compartir snippets |
| pkg.go.dev | [pkg.go.dev](https://pkg.go.dev) | Documentación de cualquier paquete de la std lib o externo |
| Go by Example | [gobyexample.com](https://gobyexample.com/) | Ejemplos concisos y ejecutables de cada concepto del lenguaje |

---

## 🌐 Artículos recomendados

- **[How to Write Go Code](https://go.dev/doc/code)** — go.dev
  Guía oficial sobre cómo organizar módulos, paquetes y workspace

- **[Go Proverbs](https://go-proverbs.github.io/)** — Rob Pike
  18 proverbios que resumen la filosofía del lenguaje; leer antes de la semana 01

- **[The Go Blog — Using Go Modules](https://go.dev/blog/using-go-modules)** — go.dev
  Explicación detallada de `go mod init`, `go get` y `go mod tidy`

---

> **Prioridad**: empieza por A Tour of Go (30-45 min) y Go Proverbs (10 min).
> El resto son referencias para cuando tengas preguntas específicas.
