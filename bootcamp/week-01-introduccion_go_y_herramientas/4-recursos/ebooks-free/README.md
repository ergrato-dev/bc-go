# Recursos — Ebooks Gratuitos

Libros en acceso libre recomendados para complementar la Semana 01.

---

## 📘 The Go Programming Language (Donovan & Kernighan)

- **Enlace**: [gopl.io](https://www.gopl.io/) — Código fuente y ejemplos gratuitos
- **Capítulos relevantes esta semana**: Cap. 1 (Tutorial), Cap. 10 (Packages and the Go Tool)
- **Por qué leerlo**: es el libro de referencia más citado en la comunidad Go; escrito por un co-autor del lenguaje C

---

## 📗 Go Bootcamp (Matt Aimonetti)

- **Enlace**: [miek.nl/go/](https://miek.nl/go/) — HTML gratuito
- **Capítulos relevantes esta semana**: Introduction, Basics
- **Por qué leerlo**: introducción concisa y práctica; ideal como segundo material de lectura

---

## 📙 An Introduction to Programming in Go (Caleb Doxsey)

- **Enlace**: [www.golang-book.com/books/intro](https://www.golang-book.com/books/intro)
- **Capítulos relevantes esta semana**: Cap. 1-3
- **Por qué leerlo**: muy accesible para principiantes absolutos; explicaciones paso a paso

---

> **Nota**: no es obligatorio leer todos estos libros. Elige uno como referencia
> principal y úsalo para profundizar cuando la teoría del bootcamp no sea suficiente.
