# Semana 01 — Introducción a Go y Herramientas

## 📋 Descripción

Primera semana del bootcamp. Instalamos Go, exploramos el toolchain oficial, entendemos el modelo mental del lenguaje y escribimos nuestro primer programa idiomático.

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Instalar Go y verificar el entorno de desarrollo
- ✅ Usar los comandos esenciales del CLI de Go (`go run`, `go build`, `go fmt`, `go vet`)
- ✅ Entender la estructura de un programa Go (`package`, `import`, `func main`)
- ✅ Explicar el modelo de compilación y el sistema de tipos de Go
- ✅ Crear y ejecutar un módulo Go con `go mod init`
- ✅ Configurar VS Code con la extensión oficial de Go

## 📚 Requisitos Previos

- Computadora con acceso a internet
- Cuenta en GitHub
- VS Code instalado
