# Semana 04 — Funciones

> **Bootcamp Go Zero to Hero** · Semana 4 de 20

---

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Definir funciones con **múltiples valores de retorno** y retornos nombrados
- Escribir funciones **variádicas** y comprender cuándo usarlas
- Tratar funciones como **valores de primera clase** (asignar, pasar, retornar)
- Construir y usar **closures** para capturar estado de forma segura
- Aplicar **funciones de orden superior** (`map`, `filter`, `reduce` idiomáticos en Go)
- Identificar los **errores comunes** al trabajar con closures y variádicas

---

## 📚 Requisitos previos

- Semana 01: variables, tipos básicos, `fmt`
- Semana 02: constantes, `iota`
- Semana 03: `if`, `for`, `switch`, `defer`
