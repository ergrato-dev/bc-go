# Ebooks Gratuitos — Semana 05: Arrays y Slices

## Recursos recomendados

### The Go Programming Language (Donovan & Kernighan)

- **Capítulo 4** — Composite Types: Arrays, Slices, Maps y Structs
- Disponible en: https://www.gopl.io/
- Cubre en profundidad la estructura interna de slices y el comportamiento de append

### Go Bootcamp (Matt Aimonetti)

- **Capítulo 4** — Collection Types
- Disponible en: http://www.golangbootcamp.com/book/collection_types
- Explica arrays y slices con ejemplos progresivos

### An Introduction to Programming in Go (Caleb Doxsey)

- **Capítulo 6** — Arrays, Slices y Maps
- Disponible en: https://www.golang-book.com/books/intro/6
- Introducción accesible con ejemplos del mundo real

### Go 101

- Sección: Containers in Go
- Disponible en: https://go101.org/article/container.html
- Referencia completa sobre arrays, slices y maps con detalles de implementación
