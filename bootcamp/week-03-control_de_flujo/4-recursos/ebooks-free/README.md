# Recursos — Ebooks Gratuitos

Libros en acceso libre recomendados para complementar la Semana 03.

---

## 📘 The Go Programming Language (Donovan & Kernighan)

- **Enlace**: [gopl.io](https://www.gopl.io/)
- **Capítulos relevantes esta semana**: Cap. 1 (Tutorial — hello world usa for y switch), Cap. 5 (Functions — defer)
- **Por qué leerlo**: la cobertura de `defer` en el Cap. 5 es la más completa disponible en formato libro

---

## 📗 Go Bootcamp (Matt Aimonetti)

- **Enlace**: [miek.nl/go/](https://miek.nl/go/)
- **Capítulos relevantes esta semana**: Control Structures, Functions (defer)
- **Por qué leerlo**: ejemplos concisos de todos los patrones de control de flujo

---

## 📙 An Introduction to Programming in Go (Caleb Doxsey)

- **Enlace**: [www.golang-book.com/books/intro](https://www.golang-book.com/books/intro)
- **Capítulos relevantes esta semana**: Cap. 5 (Control Structures)
- **Por qué leerlo**: cubre `for`, `if` y `switch` con ejemplos fáciles de seguir para principiantes

---

> **Nota**: para `defer` avanzado (panic/recover), el artículo del blog oficial es la referencia más directa.
