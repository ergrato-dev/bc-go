# Recursos — Ebooks Gratuitos

Libros en acceso libre recomendados para complementar la Semana 02.

---

## 📘 The Go Programming Language (Donovan & Kernighan)

- **Enlace**: [gopl.io](https://www.gopl.io/)
- **Capítulos relevantes esta semana**: Cap. 3 (Basic Data Types), Cap. 2 (Program Structure — declaraciones, var, const)
- **Por qué leerlo**: el capítulo 3 es la referencia más completa de tipos básicos; incluye los detalles de tamaño, overflow y conversión

---

## 📗 Go Bootcamp (Matt Aimonetti)

- **Enlace**: [miek.nl/go/](https://miek.nl/go/)
- **Capítulos relevantes esta semana**: Types, Variables, Constants
- **Por qué leerlo**: explicaciones concisas con ejemplos directamente ejecutables

---

## 📙 An Introduction to Programming in Go (Caleb Doxsey)

- **Enlace**: [www.golang-book.com/books/intro](https://www.golang-book.com/books/intro)
- **Capítulos relevantes esta semana**: Cap. 3 (Types), Cap. 4 (Variables)
- **Por qué leerlo**: muy accesible; explica bien por qué Go no permite conversiones implícitas

---

> **Nota**: el capítulo de `iota` no tiene tratamiento exhaustivo en libros introductores.
> Para patrones avanzados de `iota`, consulta directamente la [especificación de Go](https://go.dev/ref/spec#Iota).
