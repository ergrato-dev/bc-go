# Recursos — eBooks Gratuitos

## Métodos e Interfaces en Go

### The Go Programming Language (Donovan & Kernighan)
- **Capítulo 6** — Métodos: receptores, method sets, encapsulación
- **Capítulo 7** — Interfaces: satisfacción implícita, `io.Writer`, `fmt.Stringer`, `error`
- Considerado la referencia definitiva sobre interfaces en Go

### Go Bootcamp (Matt Aimonetti)
- **Capítulo: Methods** — métodos sobre tipos y structs
- **Capítulo: Interfaces** — duck typing y satisfacción implícita
- https://www.golangbootcamp.com/book/methods

### An Introduction to Programming in Go (Caleb Doxsey)
- **Capítulo 9: Structs and Interfaces** — cubre satisfacción implícita
- https://www.golang-book.com/books/intro/9

### Learning Go (Jon Bodner) — O'Reilly
- Capítulo sobre interfaces con énfasis en "accept interfaces, return structs"
- Disponible con prueba gratuita de O'Reilly
