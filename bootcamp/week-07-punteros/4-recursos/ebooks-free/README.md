# Recursos — eBooks Gratuitos

## Punteros y Gestión de Memoria en Go

### The Go Programming Language (Donovan & Kernighan)
- **Capítulo 2.3** — Variables y punteros
- **Capítulo 4.1** — Arrays (paso por valor)
- Disponible para leer online en varios sitios educativos con acceso libre
- Referencia de calidad editorial sobre punteros idiomáticos en Go

### Go Bootcamp (Matt Aimonetti)
- **Capítulo: Pointers** — explicación básica de `&` y `*`
- Disponible en: https://www.golangbootcamp.com/book/pointers

### An Introduction to Programming in Go (Caleb Doxsey)
- **Capítulo 8: Pointers** — cubre `*`, `&`, y paso por referencia
- Disponible en: https://www.golang-book.com/books/intro/8

### Learning Go (Jon Bodner)
- Sección sobre gestión de memoria y cuándo usar punteros
- Disponible en O'Reilly con cuenta de biblioteca o prueba gratuita
